Objetivo:
Comprender y aplicar los diferentes tipos de datos disponibles en Java.
Practicar la realización de operaciones aritméticas y lógicas utilizando los tipos de datos
adecuados.
Dominar el uso de las estructuras de control condicionales (if) para la toma de decisiones
en un programa.
Familiarizarse con las estructuras de control iterativas (while y for) para la repetición
controlada de instrucciones.

Temario:
1. Ingresar la nota de una determinada materia. El sistema debe decir que si el alumno tiene
   una nota menor a cuatro desaprobó la materia, si el alumno tiene una nota mayor a 4
   aprobó y si aprueba con 10 el alumno es sobresaliente.
2. Un alumno rinde tres parciales de una determinada materia. Hacer un sistema que diga si
   el alumno esta promocionado o no. Para promocionar el alumno debe tener un promedio
   de 8.
3. Se ingresa por teclado un número positivo de uno o dos dígitos (1..99). Se debe mostrar un
   mensaje indicando si el número tiene uno o dos dígitos.
4. Se carga una fecha (día, mes y año) por teclado. Mostrar un mensaje si corresponde al
   primer trimestre del año (enero, febrero o marzo), al segundo trimestre, al tercer trimestre
   o al cuarto trimestre. Cargar por teclado el valor numérico del día, mes y año.
   Ejemplo: dia:8 mes:4 año:2024.
5. Se debe ingresar la coordenada de un punto en el plano, es decir dos valores enteros x e y
   (distintos a cero). Posteriormente se debe imprimir en pantalla en que cuadrante se ubica
   dicho punto. (1o Cuadrante si x > 0 e y > 0 , 2o Cuadrante: x < 0 e y > 0, etc.)
6. Realizar un sistema que lea los lados de n triángulos, e informar:
   a) De cada uno de ellos, qué tipo de triángulo es: equilátero (tres lados iguales), isósceles
   (dos lados iguales), o escaleno (ningún lado igual)
   b) Cantidad de triángulos de cada tipo.
   c) Tipo de triángulo que posee menor cantidad.
